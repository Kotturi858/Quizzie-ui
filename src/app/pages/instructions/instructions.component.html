Instructions:

  1. This quiz has 2 modes:
    1. Competitive Mode
    2. Practise Mode
  2. In Competitive mode, you will be having a time limit and fixed number of questions and will 'not be Allowed' to go back once saved
  3. In Practise Mode of Quiz, you will be having no time limit and given a choice of choosing any question at any point of time.
  4. Your score will be calculated based only on the attempted Competitive mode questions. Please, Select Competitive mode if you want to increase the score. To understand how the score is calculated, please check the FAQs in the 'Help' section
  5. No variation in score for the practise mode attempted
  6. The score is exponential in nature, depending upon the 3 factors: Time takenn, Number of correct questions, Number of Incorrect Questions
